 public class IdentifyMyParts {

        public static int x = 7;
        public int y = 3;
}
   1) What are the class variables?

        Static variables are the class variables. Because the class variable value is changed by the class level. It consumes the class memory.

   2) What are the instance variables?

        Non static variables are the instance variables. Because the values are changed based on the object level. It consumes the object memory.
